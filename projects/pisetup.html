Pi Setup

An overview of my current raspberry pi setup, and the steps taken to get it running.

10/21/2025

My raspberry pi is going to be used as a network and monitoring hub that will later be coupled with a seperate mini pc that will act as a data/media center for my home.

OS: Raspberry Pi OS Lite (64-bit)

Services:

  • Adguard Home: Network-wide ad & tracker blocking.
  • Uptime Kuma: Will provide service monitoring for the seperate NAS device.
  • WireGuard VPN: Allows remote secure access to my home network.
  • Syncthing: Receive backups from my NAS device, only for photos, documents, and notes.

To start you will need a 16GB+ SD card to house the operating system. If your computer doesn't have an SD card slot you'll need a SD to USB product. You will also need to download the Raspberry Pi Imager and set up a user and enable ssh.
